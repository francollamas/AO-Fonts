# AO-Fonts
Editor de fonts.ind para AO

///// AGREGAR UNA NUEVA FUENTE /////

Instrucciones:
1. Abrir el programa CBFG (Codehead’s Bitmap Font Generator).
Funciona solo en Windows

2. Armar la fuente a gusto

3. Exportar un PNG y un CSV

4. Abrir la app de AO Fuentes
Esta app de consola permite ver, guardar, modificar y borrar las fuentes en un archivo 'fonts.ind' que el AO reconoce.

5. Una vez terminado de cofigurar, poner 'fonts.ind' en la carpeta Init del AO.

6. Agarrar los PNG antes creados, cambiarles el fondo negro por transparente, convertir a potencias de 2 CUADRADAS. (256x256, 512x512, etc)

7. Cambiar el nombre de esos PNG por los correspondientes

8. Ponerlos en la carpeta Graficos.

9. En VB6, al diubujar los textos, elegir la fuente correspondiente, partiendo desde el 1
